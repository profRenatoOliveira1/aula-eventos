Eventos de Teclado

Evento Quando é disparado Exemplo
onkeydown Quando uma tecla é pressionada Detectar atalho
onkeyup Quando uma tecla é solta Mostrar o que foi digitado
onkeypress Quando uma tecla é pressionada e mantida Validação de entrada

Exemplo: onkeydown

Pressione qualquer teclado dentro do input para ver o que acontece

Dica: mantenha a tecla pressionada por um tempo, depois solte para ver o que acontece.

Exemplo: onkeyup
