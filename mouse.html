Evento Quando é disparado Exemplo
onclick Quando o usuário clica sobre o elemento Botão que mostra uma mensagem
ondblclick Quando o usuário usa o clique duplo do mouse Ativar/desativar algo
onmouseover Quando o pointeiro passa em cima do elemento Destacar um botão
onmousein Quando o ponteiro do mouse entra no elemento Destacar um botão
onmouseout Quando o ponteiro do mouse sai do elemento Retirar destaque
onmousedown Quando o botão do mouse é pressionado Iniciar um arraste de elemento
onmouseup Quando o botão do mouse é solto Fim de um arraste
